# Collision Data Get Player

Gets player of a collision data (or null for objects) returned from "Overlap Sphere" chip.

| Input Name | Input Type |
|-----------|-----------|
| Target | collision data |

| Output Name | Output Type |
|-----------|-----------|
| Player | player |
