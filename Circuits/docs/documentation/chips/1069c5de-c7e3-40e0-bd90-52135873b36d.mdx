# Quaternion Create From To

Creates a rotation which rotates from "From" to "To".



![Example banner](./assets/1069c5de-c7e3-40e0-bd90-52135873b36d.svg)

| Input Name | Input Type |
|-----------|-----------|
| From | vector3 |
| To | vector3 |

| Output Name | Output Type |
|-----------|-----------|
| Result | quaternion |
